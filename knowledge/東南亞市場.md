目標市場： 核心為東南亞市場（包含柬埔寨、印度等），並需相容歐美市場標準。東南亞市場極度注重「文化敏感度」與「服裝尺度」，例如海灘比基尼主題需在保持活力的同時，尊重當地「端莊 (Modesty)」的民情，避免過度裸露與色情表現
UI 風格特徵（色調、強調色、字體風格、圖標風格等）
色調計畫 (Color Palette)：
避免使用過於老舊的紅藍搭配，尋求現代感對比
印度或東南亞風格大廳，若整體偏綠色，應加入紫、粉紅或紅色的在地化圖騰或裝飾點綴以打破單調
不重要的背景必須降底飽和度或模糊化 (Blur)，讓焦點集中於前景與主要 UI
強調色 (Accent Colors)：
購買按鈕 [Buy] 統一強烈建議使用綠色，符合手機遊戲通用習慣，使其在畫面中具備最高凸顯度
嚴禁在紅色背景上使用粉紅色文字，禁止在黃色背景上使用黃色文字，確保高度視覺對比
字體風格 (Typography)：
字體必須清晰易讀，嚴禁加上外發光暈 (Outer Glow)，外光暈會導致字體模糊、難以閱讀
字樣設計應扁平化，禁止做成過度複雜的「包黑框凹陷感」，漸層必須簡化
文字的傾斜角度不可大於 30 度
所有介面與轉場的免費遊戲次數，一律統一標示為 Free Spins (而非 Free Games)
圖標風格 (Icon Style)：
圖形視覺平衡： 並排的 UI Icon 若形狀不同，為彌補方形的視覺佔比優勢，圓形和三角形的圖標應手動放大約 5%
光源統一： 操作介面圖標需有明確的主光源與暗部次要光源；同類型的圖標（如底部 Bar、選單）使用的材質最多不超過三種，且光源方向與大小必須一致
設計規範與原則（顧問建議的具體規則，例如：對稱性、顏色使用、文字重疊標準、背景明暗等）
排版與對齊：
為減輕玩家閱讀負擔，每行文字應控制在 15 個字以內
圓角框內的元素對齊應從「圓角內部」邊緣開始計算；提示符號（如 [i] 或 [?]）應放置在對齊線之外
背景明暗與空間感 (Depth & Lighting)：
彈窗防干擾機制： 當彈出視窗 (Popups) 出現時，後方的所有背景與物件必須加上深色遮罩壓暗 (Darken everything behind)，以降低視覺干擾
轉輪盤面 (Reels)： 轉輪的上下邊緣必須壓暗（加入陰影或深色漸層），以創造出轉動的 3D 空間深度
Symbol 立體感： 所有的 Symbol 需在底部加上地推陰影 (Floor shadow)，並在後方添加背光 (Backlight glow)，使其從背景中立體跳脫出來
光影原則： 主要角色或畫作必須設定「一個主光源 (Key light)」以及「一到兩個輔助光（如 Rim light 邊緣光、Warm fill 補光）」，避免暗部過於平淡
文字重疊標準 (Overlap)：
文字與其他物件的重疊必須經過深思熟慮。若不自然地卡在中間會造成視覺緊張。解決方案有三：完全分開並留出空間、有目的地大膽斜放覆蓋、或替文字加上獨立的徽章底板與背景隔開
彈出視窗內的訊息文字下方，建議加上深色底板以保護閱讀性
TADA 公版與客製化規範：
使用 TADA 拼版公版時，機制按鈕只需保留最上方兩顆
SPIN 按鈕為絕對的視覺焦點，其他功能小按鈕應弱化（半透或少許材質），禁止使用過多無意義的框線與整塊大底板設計
完稿與技術規範：
數位介面需為 RGB 模式，符合 Retina 高解析度標準，確保邊緣無像素碎裂 (Fractional Pixels)
向量文字與標誌在交付前需建立外框 (Outline)，並確保四周留足 3mm 出血 (Bleed)
UI 設計底圖以 1136x640 範圍內顯示為主，直版需左右最大化撐滿版面
常見設計問題與退回原因
視覺過度雜亂 (Visual Clutter)： 彈出視窗中使用了過多無意義的框線、層疊的盒子（框裡有框）與分隔線。顧問強烈要求移除多餘的標題（例如在內部視窗重複出現遊戲名稱）與框線，改用陰影和顏色來區隔資訊塊
贅字過多： 按鈕上放置了不必要的動作說明（例如在 [Buy] 按鈕上還寫 "Click to buy" 或在彈窗寫 "Click a Ticket to change"），浪費空間且不夠直覺，應直接刪除
主次不分，搶奪焦點： 背景的動態、特效或高光過亮，導致前景的中獎金額、商品或主 Symbol 被吃掉。不重要的背景必須降低飽和度或模糊化
高低階 Symbol 差異不足： 低賠率 (Low Pay) Symbol 加上了過於複雜的背景、材質或發光效果，導致與高賠率 (High Pay) Symbol 視覺強度混淆。低階物件應移除背景，保持單純與乾淨
文字閱讀性差： 大獎 (Jackpot) 金額對比度太低，或是使用過度扭曲的 3D 透視文字（如底部過厚），導致俗氣且無法快速閱讀
材質與透視不統一： 畫面中同時出現扁平插畫與高度 3D 渲染的物件，或麻將、撲克牌等物件的翻轉透視角度不自然（例如只看到正面，缺乏底部的立體厚度與陰影）
